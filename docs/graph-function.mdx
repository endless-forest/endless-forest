## **Functions Graphed**

### **$y = ax + k$** 
  **Parent Function**: $y = x$
- $y=2x+1$
    - **vertical factor:** $2$
    - **vertical shift:** $1$ unit up
- $y=2x+2$
    - **vertical factor:** $2$
    - **vertical shift:** $2$ units up
- $y=2x+3$
    - **vertical factor:** $2$
    - **vertical shift:** $3$ units up
- $y=2$
    - **vertical factor:** $0$
    - **vertical shift:** $2$ units up
- $y=3$
    - **vertical factor:** $0$
    - **vertical shift:** $3$ units up
- $x=2$
    - **vertical factor:** $undefined$
    - **horizontal shift:** $2$ units right
- $x=3$
    - **vertical factor:** $undefined$
    - **horizontal shift:** $3$ units right

### **$y = a(x - h)^2 + k$**
  **Parent Function**: $y = x^2$
  - $y=10(x-5)^2 + 3$
     - **vertical factor:** $10$
     - **vertical shift:** $3$ units up
     - **horizontal shift:** $5$ units right
  - $y=-5(x-6.25)^2 + 6$
     - **vertical factor:** $-5$
     - **vertical shift:** $6$ units up
     - **horizontal shift:** $6.25$ units right
  - $y=10(x-7.5)^2 + 3$
     - **vertical factor:** $10$
     - **vertical shift:** $3$ units up
     - **horizontal shift:** $7.5$ units right

### **$y = \sqrt{x - h} + k$**
  **Parent Function**: $y = \sqrt{x}$
  - $y = \sqrt{x + 8} + 2.3$
     - **vertical factor:** $1$
     - **vertical shift:** $2.3$ units up
     - **horizontal shift:** $8$ units left
  - $y = -\sqrt{x + 8} + 1.6$
     - **vertical factor:** $-1$
     - **vertical shift:** $1.6$ units up
     - **horizontal shift:** $8$ units left
  - $y = \sqrt{x + 8.2} + 1.6$
     - **vertical factor:** $1$
     - **vertical shift:** $1.6$ units up
     - **horizontal shift:** $8.2$ units left

### **$y = (x - k)^2 + (y - k)^2 = r^2$**
  **Parent Function**: $x^2 + y^2 = r^2$
   - $x^2 + y^2=4$
     - **vertical shift:** $0$ units
     - **horizontal shift:** $0$ units
   - $y = (x-2)^2 + (y+2)^2=4$
     - **vertical shift:** $2$ units down
     - **horizontal shift:** $2$ units right

### **$y = a_{n}x^{n} + a_{n-1}x^{n-1} + a_{n-2}x^{n-2} + ... + a_{1}{x} + a_{0}$**
   - $y = x^4 + x^3 + x^2 + 4$
   - $y = x^7 + x^3 - 18$

### **$y = \frac{P(x)}{Q(x)}$**

  - $y = (x+7)^{2}/2x - 1$
  - $y = -(x+7)^{2}/2x - 1$
